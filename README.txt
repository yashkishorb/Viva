# Auto-Downloader Server

When someone opens the URL in a browser, all files are instantly downloaded as a ZIP.

## Setup

1. Install dependencies (only once):
   npm install

2. Put YOUR code files inside the `/files` folder
   (delete example.txt, add your .py / .java / .c / .txt files)

3. Start the server:
   node server.js

4. Open in browser:
   http://localhost:3000

→ The ZIP file will download automatically. No clicks needed.

## To share with others on the same network:
   Find your IP address (ipconfig / ifconfig), then share:
   http://YOUR_IP:3000
